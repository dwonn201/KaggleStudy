**Kaggle 데이터 분석 - 유럽 축구 승패예측**


European Soccer Database
데이터 출처: https://www.kaggle.com/hugomathien/soccer¶
database.sqlite 데이터베이스를 사용

데이터 베이스 내 총 7개의 Table 사용

Country: 국가 정보  
League: 리그 정보  
Match: 경기 정보 (주 데이터셋)  
Player: 플레이어 정보  
Player_Attributes: 플레이어의 특성  
Team: 팀 정보  
Team_Attributes: 팀의 특성
